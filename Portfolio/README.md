 Pragadeeswaran Portfolio

This is my personal portfolio website built using **HTML**, **CSS**, and **JavaScript**. I developed this project to showcase my skills, projects, education, certifications, internships, and achievements in **Web Development, IoT, Embedded Systems, and Software Development**.

The portfolio serves as my professional digital identity, enabling recruiters, collaborators, and peers to explore my work, understand my technical expertise, and connect with me through various platforms.

---

 About the Project

I wanted to build a portfolio that reflects both my technical abilities and my passion for creating modern web interfaces. Instead of using a conventional portfolio layout, I designed the website with a clean dark theme, smooth animations, modern UI components, and an intuitive navigation experience.

The portfolio introduces who I am, highlights my technical skills, showcases my featured projects, presents my educational background, certifications, internships, and provides multiple ways to connect with me.

Throughout this project, I focused on creating a responsive, visually appealing, and user-friendly experience while following modern frontend development practices.

1. Download or clone the repository.
2. Open the project folder.
3. Ensure all assets are placed in their respective directories.
4. Open **index.html** in any modern web browser.

No additional installations or external libraries are required.
