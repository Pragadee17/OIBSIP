🌡️ Temperature Converter

A simple, responsive, and user-friendly **Temperature Converter** built using **HTML5**, **CSS3**, and **JavaScript**. This project was developed as part of the **Web Development Internship Program at Oasis Infobyte (OIBSIP)** to strengthen my JavaScript fundamentals and frontend development skills.

The application enables users to quickly convert temperatures between **Celsius (°C)** and **Fahrenheit (°F)** with accurate calculations and a clean, interactive interface.

---

📌 About the Project

The Temperature Converter is a frontend web application designed to perform instant temperature conversions between Celsius and Fahrenheit. The goal of this project was to gain practical experience with JavaScript by implementing real-time calculations, input validation, and DOM manipulation while creating a simple and intuitive user experience.

The application features a minimal yet modern interface that focuses on usability, allowing users to enter a temperature value, choose the conversion type, and instantly view the converted result.

This project helped me understand how JavaScript interacts with HTML elements, handles user inputs, validates data, and updates webpage content dynamically.

---

 ✨ Features

* 🌡️ Convert **Celsius to Fahrenheit**
* 🌡️ Convert **Fahrenheit to Celsius**
* ⚡ Instant and accurate temperature conversion
* 🎨 Clean and responsive user interface
* ✅ Input validation for better user experience
* 🔄 Supports both positive and negative temperature values
* 📱 Mobile-friendly responsive layout
* 💻 Built entirely using HTML5, CSS3, and JavaScript

 🚀 How to Use

1. Enter a temperature value.
2. Select the conversion type:

   * Celsius to Fahrenheit
   * Fahrenheit to Celsius
3. Click the **Convert** button.
4. The converted temperature will be displayed instantly.

---

 ✅ Validation

The application includes basic input validation to improve user experience.

* Prevents empty input values.
* Accepts both positive and negative temperature values.
* Displays an appropriate message for invalid inputs.
