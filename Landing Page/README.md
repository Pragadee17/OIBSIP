 🍕 PizzOra – Modern Pizza Delivery Landing Page

A modern, responsive, and visually appealing pizza delivery landing page built using **HTML5** and **CSS3**. This project was developed as part of the **Web Development Internship Program at Oasis Infobyte (OIBSIP)** to strengthen my frontend development and UI/UX design skills.

---

 📌 About the Project

PizzOra is a modern landing page designed for a fictional pizza delivery brand. The primary objective of this project was to create a clean, engaging, and user-friendly interface while applying responsive web design principles and industry-standard frontend development practices.

The website features a visually engaging hero section, featured menu cards, service highlights, an order tracking interface, customer testimonials, and a professional footer. Every section is carefully structured to provide a seamless browsing experience and encourage user interaction through clear call-to-action elements.

This project helped me improve my understanding of semantic HTML, responsive layouts, CSS styling, typography, spacing, and designing professional landing pages that prioritize both aesthetics and usability.
